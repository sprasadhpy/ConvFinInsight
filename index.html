About the Project

ConvFinBot answers real-world financial questions based on the ConvFinQA dataset using language models.

  • Data Source: ConvFinQA
  • Technology: LLM-powered conversational AI
  • Purpose: To provide accuratefinancial answers to user queries

    • This project demonstrates an agentic financial question answering system designed for faithful, multi-step reasoning over semi-structured financial data. Key components include:

    • Data and Domain Understanding: Incorporates descriptive and summary-level statistics from the ConvFinQA dataset which is enriched with financial document context to ground agent behavior in real-world reporting structures.
    • Agent Architecture: Built on top of GPT-3.5 and Unsloth/Qwen1.5-14B the agent engages in multi-turn financial dialogues with capabilities tailored for symbolic and arithmetic reasoning over time-dependent financial variables.
    • Hybrid Reasoning Stack: Integrates Chain-of-Thought (CoT), Faithful CoT (natural language → symbolic chain execution), and a calculator augmented execution engine to bridge language-level reasoning and numerate fidelity.
    • Conversational Evaluation: Benchmarked on over 200 sampled and synthesized financial dialogues using hybrid CoT and extended symbolic reasoning and grounded by intermediate program verification and trace validation using Langchain.
    • Prompting Strategy Benchmark: Explores diverse prompting schemas—zero-shot, CoT, tool-augmented prompts—logged via LangChain instrumentation to trace reasoning fidelity and deviation patterns in financial statement.
    • Self-Evaluation Module: Embeds a reflective agent layer for post-hoc verification of answer consistency and reasoning trace alignment, enabling adaptive correction in the loop.
    • Extended Inference Control: Leverages <WAIT> tokens and recent s1 delayed decision primitives to support test-time scaffolding technique.
    • External Knowledge Alignment: Integrates FMP API to inject real-time financial data : facilitating hallucination avoidance and also analysis of agentic failure modes under partial observability.
    • Comparative Model Evaluation: Conducts targeted accuracy assessments between GPT-3.5 and fine-tuned Unsloth Qwen1.5-14B across multi-hop financial reasoning scenarios.
    • Optimization Techniques: Applies Supervised Fine-Tuning (SFT) and Group Relative Policy Optimization (GRPO) via Unsloth to improve trajectory faithfulness under reward-aware alignment signals.
    • Retrieval-Augmented : Implements simple RAG pipelines for supporting contextual grounding enabling the agent to condition on structured and semi-structured financial narratives during answer generation.
